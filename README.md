PVZTowers
</h1>

A Mod That Adds Plants From PVZ to the game. Also has a plant food mechanic that applies a buff that only works on plants and does different things depending on the plant. Planned Updates: Definitely new plants, and maybe pvz boss bloons! There is a bug with the plant food where sometimes it does nothing the first time it is applied hopefully that will be fixed soon as well.
Now has a setting to turn on PVZ mode allowing the plants to be plced on the track to block bloons and take damag while blocking them. This allows it to feel more like the PVZ games.
